1. ¿Por qué aprender Python?

El lenguaje del futuro

Python es un lenguaje de programación de alto nivel, interpretado y con una sintaxis muy limpia.

Sus principales usos incluyen:

  • Ciencia de Datos y Análisis.
  • Inteligencia Artificial y Machine Learning.
  • Desarrollo Web (Backend).
  • Automatización de tareas (Scripting).

2. Configurando el Entorno

Sigue estos pasos para preparar tu computadora:

  1. Descarga la última versión desde python.org.
  2. Ejecuta el instalador y marca la casilla "Add Python to PATH".
  3. Verifica la instalación abriendo una terminal y escribiendo:
python --version

Pro-tip: Te recomendamos usar Visual Studio Code como editor de texto.

3. Fundamentos de Programación

Variables y Tipos de Datos

En Python no necesitas declarar el tipo de variable, se asigna automáticamente.

# Ejemplo de código
nombre = "Amigo"
edad = 25
es_estudiante = True
precio = 19.99

Operadores Matemáticos

Operación Símbolo Ejemplo
Suma + 5 + 2 = 7
Resta - 10 - 4 = 6
Multiplicación * 3 * 3 = 9
División / 10 / 2 = 5.0
Potencia ** 2 ** 3 = 8

Condicionales (If-Else)

Controla el flujo de tu programa basado en decisiones.

if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")

4. Organizando la Información

Listas

Colecciones ordenadas y mutables.

  • frutas = ["manzana", "banana", "cereza"]
  • Acceso por índice: frutas[0]

Diccionarios

Almacenan datos en pares de Clave-Valor.

usuario = {
    "nombre": "Juan",
    "email": "juan@example.com"
}

5. Repetición de Tareas (Loops)

Python utiliza principalmente dos tipos de bucles:

Bucle For

Ideal para recorrer colecciones.

for i in range(5):
    print(f"Número: {i}")

Bucle While

Se ejecuta mientras una condición sea verdadera.

contador = 0
while contador < 3:
    print(contador)
    contador += 1

6. Funciones: Código Reutilizable

No te repitas (DRY - Don't Repeat Yourself).

def saludar(nombre):
    return f"Hola, {nombre}!"

mensaje = saludar("Mundo")
print(mensaje)
